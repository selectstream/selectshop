# SelectStream Shop

Boutique commerce engine powered by Astro 5.0 and Notion.

## Features
- **High-Performance SSR:** Optimized for global delivery on Netlify.
- **Decoupled Content:** Headless architecture via Astro Content Layer.
- **Compliance First:** Integrated FTC and affiliate integrity systems.

## Local Setup
1. Clone the repository.
2. Configure `.env` (Notion & Affiliate keys).
3. `npm install` && `npm run dev`.
